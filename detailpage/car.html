Entrepreneurial Development for Young Professionals in Energy

Course Overview: The machine learning course will cover several important topics such as the motivation and current state-of-the-art applications of machine learning. Participants will also learn the basics of developing a complete machine-learning project lifecycle from data collection to project deployment. Familiarity with machine learning libraries and packages will also be emphasized. The course will specifically focus on supervised machine learning applications in classification and regression problems. Participants will be able to apply the concepts learned to a real-life case study involving the oil and gas industry where they will build and deploy machine learning models.
